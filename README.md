# Amato Allestimenti Veicoli

Landing page ufficiale per Amato Allestimenti Veicoli, azienda di Nocera Inferiore specializzata dal 1965 in allestimenti su misura.

## Contenuti

- Hero con richiesta di contatto e WhatsApp diretto
- Servizi per officine mobili, Polizia Municipale, Protezione Civile e mobilità accessibile
- Configuratore interattivo per van e furgoni
- Storia aziendale e metodo di lavoro
- Layout responsive per desktop e smartphone
